# Ambulance Emergency System (AES)

A real-time emergency response platform with automated hospital allocation and dual-SLA tracking.

## 🚀 Quick Start (Localhost)

### 1. Backend Setup (Flask)
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
*Current .env points to the Railway DB.*

### 2. Frontend Setup (React + Vite)
```bash
cd frontend
npm install
npm run dev
```

---

## 📡 Deployment
- **Frontend**: https://aec-three.vercel.app
- **Backend**: Railway API endpoint
- **Database**: Remote MySQL on Railway.
